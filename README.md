# Projeto_ Emprestimo_de_Carro
Um site para coletar dados de uso de dois carros que utilizamos no trabalho.


A princípio, será um projeto simples, mas tenho a pretensão de expandi-lo. Nesse projeto, o usuário escaneará um QR Code que abrirá um site. Nesse site, haverá inicialmente uma página solicitando o nome da pessoa, com qual carro ela está saindo, a data e o motivo .
